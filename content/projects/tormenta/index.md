Tormenta provides a Scala-friendly, composable API for building Apache Storm spouts. It includes spouts for Kafka, Kestrel, Twitter's Streaming API, and other data sources. Tormenta feeds the real-time side of Summingbird's hybrid batch/streaming architecture.
