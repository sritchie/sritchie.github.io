Emmy (formerly SICMUtils) is a Clojure/ClojureScript computer algebra system implementing the math and physics tools from Sussman and Wisdom's [Structure and Interpretation of Classical Mechanics](https://mitpress.mit.edu/sites/default/files/titles/content/sicm_edition_2/toc.html) and [Functional Differential Geometry](https://mitpress.mit.edu/books/functional-differential-geometry). The library provides symbolic computation, automatic differentiation, numerical integration, and much more.

Emmy is the core of a larger ecosystem of libraries under the [mentat-collective](https://github.com/mentat-collective) organization:

- [emmy](https://github.com/mentat-collective/emmy) — core computer algebra system
- [emmy-viewers](https://github.com/mentat-collective/emmy-viewers) — 2D/3D visualization for mathematical objects
- [emmy.js](https://github.com/mentat-collective/emmy.js) — JavaScript bridge
- [MathBox.cljs](https://github.com/mentat-collective/MathBox.cljs) — WebGL math visualization
- [JSXGraph.cljs](https://github.com/mentat-collective/JSXGraph.cljs) — interactive geometry
- [Mafs.cljs](https://github.com/mentat-collective/Mafs.cljs) — 2D math visualization
- [Leva.cljs](https://github.com/mentat-collective/Leva.cljs) — GUI controls
- [MathLive.cljs](https://github.com/mentat-collective/MathLive.cljs) — equation editor
- [Clerk-Utils](https://github.com/mentat-collective/Clerk-Utils) — notebook utilities
- [functional-numerics](https://github.com/mentat-collective/functional-numerics) — numerical methods
