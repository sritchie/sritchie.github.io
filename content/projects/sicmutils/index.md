SICMUtils is a Clojure/ClojureScript library implementing the math and physics tools from Sussman and Wisdom's [Structure and Interpretation of Classical Mechanics](https://mitpress.mit.edu/sites/default/files/titles/content/sicm_edition_2/toc.html) and [Functional Differential Geometry](https://mitpress.mit.edu/books/functional-differential-geometry).

The library provides a full computer algebra system with support for symbolic computation, automatic differentiation, numerical integration, and more.
