Pallet-Hadoop automates the provisioning and configuration of Hadoop clusters using the Pallet cloud infrastructure library. It lets you spin up fully configured Hadoop clusters on AWS or bare metal with a single Clojure command, handling all the coordination of HDFS, MapReduce, and supporting services.
