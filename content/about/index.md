I'm Sam Ritchie — a programmer, mathematician, and adventurer based in Colorado.

I build open-source tools for computational mathematics and physics, run ultramarathons in the mountains, and I'm building an RV-10 airplane.

Previously, I worked at Twitter, Stripe, and the Unison Computing Company. My open-source work includes [SICMUtils](https://github.com/sicmutils/sicmutils) (a computer algebra system for physics), [Cascalog](https://github.com/nathanmarz/cascalog) (a Clojure data processing library for Hadoop), and [Summingbird](https://github.com/twitter/summingbird) (a streaming MapReduce framework).

I write about math, physics, programming, and trail running at this site. My newsletter [Road to Reality](https://roadtoreality.substack.com/) covers my journey through the mathematics of physics.
