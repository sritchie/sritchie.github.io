# Preview

Start the Hugo development server for local preview.

## Steps

1. Run `hugo server -D` to start the preview server with drafts enabled
2. Report the local URL (usually http://localhost:1313/)
3. Report any build warnings or errors
4. The server runs in the background with live reload — edits will auto-refresh
